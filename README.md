blogpy
======

A simple static blog generator in python

How to use:
<ol>
<li>git clone https://github.com/travisred/blogpy.git`</li>
<li>cd `blogpy`</li>
<li>add a markdown file to `/md`</li>
<li>`python build.py`</li>
<li>upload `/site` `/css` `/js` to host</li>
</ol>

blogpy
======

A simple static blog generator in python

How to use:
1.	`git clone https://github.com/travisred/blogpy.git`
2.	cd `blogpy`
3.	add a markdown file to `/md`
4.	`python build.py`
5.	upload `/site` `/css` `/js` to host
